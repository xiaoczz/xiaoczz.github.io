发表于 2021-08-16 | 分类于 Docker ， Docker安装

1.Docker安装

1.1.安装Docker

  • 安装Docker：yum -y install docker
  • 开机自启：systemctl enable docker
  • 启动Docker： systemctl start docker
  • 查看Docker当前的版本docker version

1.2.Docker加速仓库配置

  1. 切换到Docker的配置目录里面： cd /etc/docker/

  2. 修改daemon.json： vi daemon.json 按i编辑

    wq保存退出

  3. 配置文件修改后，重启Docker： systemctl restart docker
